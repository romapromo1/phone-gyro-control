# Архивный модуль генерации AI-образов

> Этот каталог не подключён к текущему пользовательскому сценарию. Рабочая интеграция находится в `src/experience/teamGenerationService.ts` и серверном `POST /api/generate-image`; сервер напрямую вызывает OpenRouter Images API. Не используйте описанный ниже старый пайплайн без отдельной миграции.

Этот каталог содержит вырезанный функционал генерации образов с использованием ИИ из проекта `safe-final` для интеграции в проект `phone-gyro-control`.

Код адаптирован под **Vanilla TypeScript** (без зависимостей от React и Zustand), что позволяет легко внедрить его в текущую архитектуру проекта (`desktop.ts` / `controller.ts`).

---

## 📂 Структура файлов

1. **`types.ts`**: Базовые типы данных (`Gender`, `GenerationResult`).
2. **`prompts.ts`**: Набор текстовых промптов (универсальные, мужские, женские) для генератора образов с жесткими требованиями к студийному 9:16 ракурсу и исключению артефактов.
3. **`faceCropService.ts`**: Сервис для автоматической детекции лица (использует API браузера или MediaPipe BlazeFace) и последующего кропа под пропорции 9:16. Позволяет избежать слияния лиц, если в кадре несколько человек.
4. **`generationService.ts`**: Клиентские методы для общения с бэкендом (определение пола `/api/detect-gender` и генерация стилизованного 2K изображения `/api/generate-image`). Экспортирует пайплайн `runGenerationPipeline`.
5. **`ImageGenerationUI.ts`**: Контроллер пользовательского интерфейса (Vanilla DOM). Создает:
   - Анимированный оверлей загрузки («Идёт генерация…» со спиннером).
   - Появляющийся экран успеха с кнопкой **«Посмотреть свой образ»** после окончания генерации.
   - Полноэкранный просмотрщик сгенерированного образа с кнопкой «← ЗАНОВО» и QR-кодом для скачивания (ссылка на Google Диск).

## ⚡ Быстрый старт

### 1. Установка зависимостей
Для работы сервиса кропа лиц и генерации QR-кодов требуются `@mediapipe/tasks-vision` и `qrcode`. Они уже установлены в текущем проекте.

### 2. Пример интеграции в `src/desktop.ts`

Вам достаточно импортировать класс `ImageGenerationUI` и проинициализировать его.

```typescript
import { ImageGenerationUI } from './image-generation/ImageGenerationUI';

// 1. Инициализация UI-контроллера (например, при инициализации приложения)
const imageGen = new ImageGenerationUI({
  // Контейнер, куда будут добавляться оверлеи (по умолчанию document.getElementById('app') или body)
  container: document.getElementById('app') || document.body,
  
  // Папка Google Диска, куда бэкенд загружает результаты (используется для генерации QR-кода)
  driveFolderUrl: 'https://drive.google.com/drive/folders/ВАШ_ID?usp=sharing',

  // Колбэк при клике на кнопку "← ЗАНОВО"
  onReset: () => {
    console.log('Сброс состояния игры, переход на главный экран');
    // Здесь ваша логика возврата игры в стартовое состояние
  },

  // (Опционально) Колбэк при успешном получении образа
  onSuccess: (imageUrl) => {
    console.log('Образ успешно сгенерирован:', imageUrl);
  },

  // (Опционально) Колбэк при ошибке
  onError: (error) => {
    console.error('Ошибка пайплайна генерации:', error);
  }
});

// 2. Запуск генерации (например, после подтверждения фото пользователем)
// base64Photo - снимок с веб-камеры в формате base64
function onPhotoConfirmed(base64Photo: string) {
  imageGen.startGeneration(base64Photo);
}
```

### 3. Требования к бэкенду
Для генерации используется локальный бэкенд на порту `3005` (или заданный через `import.meta.env.VITE_BACKEND_URL`).
Убедитесь, что запущен сервер, реализующий следующие API:
- `POST /api/detect-gender` (Принимает `{ imageBase64 }`, возвращает `{ gender: 'male'|'female'|'unknown' }`)
- `POST /api/generate-image` (Принимает `{ promptText, base64Image, driveFolderUrl }`, возвращает `{ imageDataUrl }` с base64-строкой готовой картинки)

## 🎨 Дизайн и анимации
Интерфейс в `ImageGenerationUI.ts` использует premium-стили:
- Шрифты из семейства `Inter` с динамическими размерами через `clamp()`.
- Изящный размытый фон (`backdrop-filter: blur(10px)`) для кнопок.
- Плавные микро-анимации появления (`ig-fadeSlideIn`) и бесконечного вращения спиннера (`ig-spin`).
- Цветовая палитра соответствует минималистичной темной эстетике (`#111` контраст на белом фоне).
