# Исправленные лабиринты с исходным профилем стен

Файлы в этой папке являются отдельными копиями исправленных моделей. Исходники из `public/source/new` и код проекта не изменялись.

В лабиринтах 2–7 сохранены исходные закругления, декоративные пояса и план проходов. Высота стен увеличена ещё на 30% — с приблизительно `1.047` до `1.361` в координатах FBX.

UV-развёртка стен пересобрана связными поверхностями, чтобы нормальная карта не проявляла диагонали при наложении полупрозрачных стен. Проверка через `THREE.FBXLoader` показывает 0 геометрических, дублирующихся и UV-вырожденных треугольников во всех моделях.
