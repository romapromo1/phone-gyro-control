# Исправленные лабиринты

Файлы в этой папке являются исправленными копиями моделей. Исходные FBX из `public/source/new` и код проекта не изменялись.

## Лабиринты 2–7

Высота стен увеличена ещё на 30% от предыдущей версии:

- предыдущая высота: около `1.047` в координатах FBX;
- новая высота: около `1.361`;
- суммарное увеличение относительно исходных стен `0.910` составляет около 49.5%;
- пол, план, ширина проходов и горизонтальные размеры не изменены;
- исходные закругления и декоративный профиль сохранены.

Для устранения треугольных артефактов полупрозрачного материала у стен заново построена связная угловая UV-развёртка. Нулевые UV-треугольники получили устойчивые локальные координаты.

Проверка через используемый приложением `THREE.FBXLoader` показывает для каждого лабиринта 2–7:

- 0 геометрически вырожденных треугольников;
- 0 дублирующихся треугольников;
- 0 UV-вырожденных треугольников.
