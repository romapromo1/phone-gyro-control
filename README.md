# Голографический 3D-Лабиринт с управлением по гироскопу (для Holobox)

Этот проект представляет собой 3D-игру лабиринт, адаптированную для вертикального 4K-экрана Holobox (соотношение сторон 9:16), управляемую в реальном времени с помощью смартфона.

---

## 🛠️ Системные требования и зависимости

1. **Компьютер с Holobox**:
   - Установлен Node.js (версия 18 или новее).
   - Компьютер и смартфон должны находиться в **одной локальной сети Wi-Fi**.

2. **Смартфон**:
   - Любой современный смартфон на iOS (Safari) или Android (Chrome) с поддержкой датчика положения (гироскопа).

---

## 🚀 Быстрый запуск

### Шаг 1. Запуск сервера
В папке проекта откройте терминал (PowerShell или CMD) и выполните:

```bash
npm run dev
```
 т
Сервер автоматически определит ваш локальный IP-адрес и запустится на порту `3000`. Вы увидите вывод в консоли:
```text
----------------------------------------------------------------
Server running at:
  - Local:      http://localhost:3000/
  - Network:    http://192.168.1.100:3000/
  - Controller: http://192.168.1.100:3000/controller.html
----------------------------------------------------------------
```

### Шаг 2. Запуск игры на Holobox (ПК)
Откройте браузер на ПК (подключенном к экрану Holobox) и перейдите по адресу:
`http://localhost:3000/` (разверните вкладку на весь экран, нажав `F11`).

### Шаг 3. Подключение смартфона
1. На экране Holobox отобразится стартовый экран с **QR-кодом** и ссылкой.
2. Отсканируйте QR-код камерой смартфона.
3. На смартфоне откроется страница контроллера.
4. Положите смартфон ровно на ладонь (экраном вверх) и нажмите кнопку **"Активировать"**.
   - *Примечание для iOS (iPhone)*: Браузер Safari запросит разрешение на доступ к датчикам движения. Подтвердите его.
5. После активации стартовый экран на Holobox мгновенно скроется, и появится 3D-лабиринт с шаром.

## 🎮 Как играть и калибровать

- **Управление**: Наклоняйте смартфон в руке. 
  - Наклон вперед опускает дальний край лабиринта (шар катится вперед).
  - Наклон влево опускает левую сторону (шар катится влево).
- **Калибровка**:
  - Если лабиринт накренился, когда вы держите телефон прямо, просто нажмите кнопку **"Калибровать"** на экране смартфона или **"Сбросить Угол"** на HUD-интерфейсе ПК. Текущее положение телефона примется за идеальный "ноль" (горизонтальное положение).
- **Физика и столкновения**: Шар реалистично катится по дорожкам лабиринта, сталкивается со стенами и отскакивает от них с учетом импульса. При падении за пределы лабиринта шар автоматически возрождается на старте.
- **Динамический звук**:
  - Качение шара сопровождается реалистичным низкочастотным гулом, громкость и частота которого зависят от скорости шара.
  - При сильных ударах о стены раздается характерный глухой стук.
- **Победа**: Докатите шар до светящейся фиолетовым цветом зоны финиша. На экране появится надпись «ПОБЕДА!» и сыграет победный синтезированный аккорд. Нажмите "Играть снова" для рестарта.

## 📁 Структура проекта

- `server.js` — Локальный веб-сервер на Express и Socket.io с интеграцией Vite-middleware (для HMR).
- `index.html` / `src/desktop.ts` — Страница игры на Holobox, рендеринг Three.js, физический движок Rapier.js (WASM) и синтезатор звуков Web Audio API.
- `controller.html` / `src/controller.ts` — Веб-интерфейс мобильного контроллера со считыванием датчиков наклона и Socket.io клиентом.
- `public/` — Директория со статическими файлами (3D-модель лабиринта `source/labirint.fbx` и PBR-текстуры в `textures/`).
- `vite.config.js` — Конфигурация Vite для компиляции многостраничного приложения.

## 💎 Оптимизация для Holobox (Голографический эффект)

1. **Черный фон**: Задний план полностью выкрашен в `#000000`. На экранах голобоксов черный цвет отображается как прозрачный, что создает иллюзию парения лабиринта внутри куба.
2. **Свечение (Bloom)**: На финише и старте установлены яркие точечные неоновые светильники (PointLight) голубого и розового оттенков, подчеркивающие глубину трехмерного пространства.
3. **PBR Материалы**: Лабиринт использует высококачественные PBR текстуры (`Normal`, `Roughness`, `Metallic`), которые красиво реагируют на источники света при изменении наклона.
