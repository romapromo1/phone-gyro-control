# Голографический 3D-Лабиринт с управлением по гироскопу (для Holobox)

Этот проект представляет собой 3D-игру лабиринт, адаптированную для вертикального 86" 4K-экрана Holobox (соотношение сторон 9:16), управляемую в реальном времени с помощью смартфона.

---

## 🛠️ Системные требования и зависимости

1. **Компьютер с Holobox**:
   - Установлен Node.js (версия 18 или новее).
   - Компьютер и смартфон должны находиться в **одной локальной сети Wi-Fi**.

2. **Смартфон**:
   - Любой современный смартфон на iOS (Safari) или Android (Chrome) с поддержкой датчика положения (гироскопа).

---

## 🚀 Быстрый запуск

### Шаг 1. Запуск сервера
В папке проекта откройте терминал (PowerShell или CMD) и выполните:

```bash
npm run dev
```

Сервер автоматически определит ваш локальный IP-адрес и запустится на порту `3000`. Вы увидите вывод в консоли:
```text
----------------------------------------------------------------
HTTPS Server running at:
  - Local:      https://localhost:3000/
  - Network:    https://192.168.1.100:3000/
  - Controller: https://192.168.1.100:3000/controller.html
----------------------------------------------------------------
```

### Шаг 2. Запуск игры на Holobox (ПК)
Откройте браузер на ПК (подключенном к экрану Holobox) и перейдите по адресу:
`https://localhost:3000/` (разверните вкладку на весь экран, нажав `F11`).

### Шаг 3. Подключение смартфона
1. На экране Holobox отобразится стартовый экран цвета `#C5DFFC` с промо-графикой, QR-кодом и дисклеймером рекламы в самом низу.
2. Отсканируйте QR-код камерой смартфона.
3. На смартфоне откроется страница контроллера.
4. Положите смартфон ровно на ладонь (экраном вверх) и нажмите кнопку **"Активировать"**.
   - *Примечание для iOS (iPhone)*: Браузер Safari запросит разрешение на доступ к датчикам движения. Подтвердите его.
5. После активации стартовый экран на Holobox мгновенно скроется, фон станет белым, и появится 3D-лабиринт с футбольным мячом.

## 🎮 Как играть и калибровать

- **Управление**: Наклоняйте смартфон в руке. 
  - Наклон вперед опускает дальний край лабиринта (мяч катится вперед).
  - Наклон влево опускает левую сторону (мяч катится влево).
- **Калибровка**:
  - Если лабиринт накренился, когда вы держите телефон прямо, просто нажмите кнопку **"Калибровать"** на экране смартфона или **"Сбросить Угол"** на HUD-интерфейсе ПК. Текущее положение телефона примется за идеальный "ноль" (горизонтальное положение).
- **Физика и столкновения**: Футбольный мяч (`football.glb`) реалистично катится по дорожкам лабиринта, сталкивается со стенами и отскакивает от них с учетом импульса. При падении за пределы лабиринта мяч автоматически возрождается на старте.
- **Динамический звук**:
  - Качение мяча сопровождается реалистичным низкочастотным гулом, громкость и частота которого зависят от скорости мяча.
  - При сильных ударах о стены раздается характерный глухой стук.
- **Победа на уровне**: Докатите футбольный мяч до золотого диска-сейва. Сейв исчезнет, и игра мгновенно перейдет на следующий уровень с анимацией прозрачности.
- **Пройти всю игру**: Соберите все **4 сейва** на 4 последовательных лабиринтах. На экране появится надпись «ПОБЕДА!» и сыграет победный синтезированный аккорд. Нажмите "Играть снова" для рестарта с 1-го уровня.

## 📁 Структура проекта

- `server.js` — Локальный веб-сервер на Express и Socket.io с интеграцией Vite-middleware (для HMR).
- `index.html` / `src/desktop.ts` — Страница игры на Holobox, рендеринг Three.js, физический движок Rapier.js (WASM) и синтезатор звуков Web Audio API.
- `controller.html` / `src/controller.ts` — Веб-интерфейс мобильного контроллера со считыванием датчиков наклона и Socket.io клиентом.
- `public/` — Директория со статическими файлами (3D-модели лабиринтов в `source/fixed/`, модель футбольного мяча `source/football.glb`).
- `vite.config.js` — Конфигурация Vite для компиляции многостраничного приложения.

## 💎 Оптимизация под 86" Вертикальный Экран

1. **Масштабируемый UI**: Все элементы интерфейса (кнопки, текст, иконки сейвов в HUD) используют относительные единицы высоты экрана (`vh`), что делает интерфейс крупным и отлично читаемым с большого расстояния.
2. **Адаптивная камера**: Расстояние до камеры автоматически пересчитывается с коэффициентом `1.9` под узкий вертикальный формат 9:16, исключая обрезание краев лабиринта.
3. **Белый фон во время игры**: Чистый белый cyclorama-фон во время геймплея создает идеальную контрастность и подчеркивает тени шара и лабиринта.
4. **Неоновый стиль кнопок**:
   - Обычный режим: фон `#C5DFFC`, обводка и текст `#080245`.
   - Наведение: фон `#DDFF68`, обводка и текст `#716CFF` с ярким неоновым свечением.
